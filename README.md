# Simple Finite State Machine (FSM) in Java

## Overview
This project demonstrates the implementation of Finite State Machines (FSM) using Java. It showcases state transitions, event handling, and state management through different real-world examples.

## Features
- Light Switch FSM
- Secure Gate FSM
- Traffic Light FSM
- State History Tracker
- Guarded Transitions with Error Handling

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Finite State Machine (FSM)

## Project Objectives
- Understand FSM concepts
- Implement state transitions in Java
- Model real-world systems using FSM
- Improve problem-solving skills
